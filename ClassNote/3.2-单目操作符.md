# 单目操作符

## `前置++ ` 和 `后置++ ` 的区别

### ⭐始终记住一句话 **从右往左**依次进行运算

```c
//单目操作符 <++，-->

//自增 ++
#include <stdio.h>
int main() {
	int a = 5;

	//前置++，先计算，后输出
	printf("%d\n",++a);
	printf("%d\n", a);

	int b = 5;
	//后置++，先输出，后计算
	printf("%d\n",b++);
	printf("%d\n", b);
	return 0;
}

//自减同理
```

```c
输出结果：
6
6
5
6
```
