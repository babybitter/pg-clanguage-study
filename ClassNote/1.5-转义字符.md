# 转义字符

一些常见的转义字符

```c
#include <stdio.h>

int main() {
	printf("\\");
	printf("\"");
	printf("\a");						//发出警告
	printf("\n");						//换行
	printf("abcdef\bg");				//退格
	printf("abcd\tefgh\tijkl\tmn");		//“相当于Tab”
	return 0;
}
```

```c
\"
abcdegabcd      efgh    ijkl    mn
```
